# HabiTribe – Social Habit Tracking Reinvented

## Overview
HabiTribe is a socially-driven habit tracking platform designed to address the key shortcomings of traditional habit tracking applications. Instead of treating habits as private activities, HabiTribe transforms them into shared experiences with accountability, structured rewards, and real consequences.

## Problem Statement
Most habit tracking applications fail due to:
- Manual logging
- Lack of social accountability
- No consequences for failure
- Low engagement

## Solution
HabiTribe introduces:
- Group-based habit tracking
- Quest-based tasks
- Reward and penalty system
- Hybrid tracking methods

## Features
- User onboarding with niche selection
- Group system (4–8 members)
- Task creation and approval
- Reward engine based on difficulty
- Proof-based validation
- Failure handling system

## Tech Stack
Frontend: React (Vite)
Backend: Node.js (Express)
Database: MongoDB / Firebase (planned)

## Getting Started

```bash
git clone https://github.com/Fluents4/Habitribe.git
cd Habitribe
cd frontend
npm install
npm run dev
```
